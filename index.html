Limitations and Suitability

This system works well for most students. It is believed to be at least as accurate as the DARS system for the progrrams it implements. (DARS implements ALL YSU programs.) For the CSIS department if this ECS shows a students has completed the requirements for a degree in the CSIS department, the department considers the requirements met (provided the correct curriculum has been used).

For a number of reasons, the ECS could produce incorrect or incomplete results. Transfer credit is particularly difficult for any automated system to handle. If something looks wrong, check with a human advisor. Humans are best equiped to handle unforeseen conditions.

Since the 2012 Gen Ed model allows some courses to meet multiple catagories, the ECS does its best to allocate such courses. It might not know your intentions for a given course. Sometimes the catagory of a course changes when other courses are added.

Privacy Policy

This system currently collects the following information:

  • Student name and banner ID.
  • The major and minor as listed on the transcript (without student information).
  • The Banner ID of the user (student or advisor) and time that the system was used.
  • Counts of how many courses of the various Gen Ed 2012 catagories are on the transcript (without any ID or grades).
  • Some debugging information such as what courses match various requirements (without any ID or grades)
  • Some debugging information such as what courses match various requirements (without any ID or grades)
Using this system implies concent for such data to be collected.

Expansion

Currently data is available for a number of major programs within the STEM college and a number of minors. Current plans for expansion include:

  • Implementing more minors
  • Implementing all majors within STEM.
  • Limiting the selection of majrs and minors to those a student can leitimately use.
